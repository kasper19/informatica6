Глоссарий

Абзац Текст, заканчивающийся знаком абзаца: ¶.
Алгоритм Понятная и конечная последовательность точных действий (команд), формальное выполнение которых позволяет получить решение поставленной задачи.
Браузер Программа, предназначенная для просмотра веб-страниц.
Буфер обмена Специальная область памяти компьютера для временного хранения данных при выполнении операций Вырезать и Копировать.
Веб-сайт Несколько веб-страниц с общей темой и содержанием, связанных между собой гиперссылками.
Веб-страница Документ или информационный ресурс Интернета.
Вспомогательный алгоритм Алгоритм, который можно целиком использовать в других алгоритмах.
Графический редактор Специальная программа, предназначенная для создания и редактирования изображений.
Графический способ записи алгоритма Запись алгоритма с помощью геометрических фигур (блоков), соответствующих командам алгоритма, и линий для соединения блоков.
Данные Сведения, представленные в формализованном виде (на одном из формальных языков).
Интернет Всемирная компьютерная сеть, способная передавать данные из одной точки земного шара в любую другую.
Информатика Наука, изучающая методы представления, хранения, передачи, обработки и использования информации с применением компьютерных технологий в различных сферах человеческой деятельности.
Информация Сведения об объектах и событиях окружающего мира.
Исполнитель алгоритма Человек, группа людей или техническое устройство, которые понимают команды алгоритма и умеют правильно их выполнять.
Команда в алгоритме Указание на выполнение конкретного действия.
Компьютер Универсальное устройство для работы с данными.
Компьютерная презентация Электронный документ, созданный на компьютере с помощью специальных программ и предназначенный для демонстрации зрителям.
Компьютерные технологии Технологии работы с данными, основанные на применении компьютеров.
Отражение изображения Зеркальное отображение изображения относительно невидимой оси (горизонтальной или вертикальной).
Презентация Способ представления информации с помощью технических средств или без них, предназначенный для демонстрации чего-либо нового (проекта, товара и т. д.).
Программное обеспечение (ПО) Совокупность всех программ, написанных человеком для компьютера.
Программный способ записи алгоритма Запись алгоритма в виде программы.
Редактирование Изменение изображения или содержания текста.
Символ текста Каждая буква, цифра, знак препинания и т. д.
Слайд Отдельная страница презентации, предназначенная для размещения материалов.
Словесный способ записи алгоритма Запись алгоритма на естественном языке общения.
Текстовый документ Документ, информация в котором представлена в виде текста.
Текстовый курсор Указатель места ввода символа.
Текстовый редактор Программа, предназначенная для работы с содержимым электронных текстовых документов.
Форматирование текста Изменение формы представления текста.
Фрагмент изображения Выделенная часть изображения.
Фрагмент текста Выделенная часть компьютерного текста.
Электронная почта Служба сети Интернет, предназначенная для обмена сообщениями (электронными письмами) с помощью электронных почтовых ящиков.
Электронный текстовый документ Текстовый документ, сохраненный на диске в файле.
Всемирная информационная паутина (WWW) служба (сервис) Интернета,позволяющий получать нужную информацию, представленную в виде специальных документов (веб-страниц, www-страниц).
Вызов процедуры команда выполнения вспомогательного алгоритма.
Гиперссылка на веб-странице часть документа (слово, предложение, рисунок), которая ссылается на другую часть этого документа или другой документ.
Компьютерный исполнитель виртуальный объект, действующий в виртуальной среде.
Программа запись алгоритма на языке программирования.
Сетевой этикет правила поведения, общения в сети, традиции и культура пользователей Интернета.
Система команд исполнителя команды, которые понимает и может выполнить исполнитель.
Среда обитания исполнителя среда, в которой действует исполнитель.
Среда программирования комплекс программ, используемых при разработке других программ.

Гласарый

Абзац Тэкст, які заканчваецца знакам абзаца: ¶.
Адбіццё відарыса Люстраное адлюстраванне відарыса адносна нябачнай восі (гарызантальнай або вертыкальнай).
Алгарытм Зразумелая і канечная паслядоўнасць дакладных дзеянняў (каманд), фармальнае выкананне якіх дазваляе атрымаць рашэнне пастаўленай задачы.
Браўзер Праграма, прызначаная для прагляду вэб-старонак.
Буфер абмену Спецыяльная вобласць памяці камп’ютара для часовага захоўвання даных пры выкананні аперацый Вырезать і Копировать.
Выканаўца алгарытму Чалавек, група людзей або тэхнічнае ўстройства, якія разумеюць каманды алгарытму і ўмеюць правільна іх выконваць.
Вэб-сайт Некалькі вэб-старонак з агульнай тэмай і зместам, звязаныя паміж сабой гіперспасылкамі.
Вэб-старонка Дакумент або інфармацыйны рэсурс Інтэрнэту.
Графічны рэдактар Спецыяльная праграма, прызначаная для стварэння і рэдагавання відарысаў.
Графічны спосаб запісу алгарытму Запіс алгарытму з дапамогай геаметрычных фігур (блокаў), якія адпавядаюць камандам алгарытму, і ліній для злучэння блокаў.
Даныя Звесткі, прыведзеныя ў фармалізаваным выглядзе (на адной з фармальных моў).
Дапаможны алгарытм Алгарытм, які можна цалкам выкарыстоўваць у іншых алгарытмах.
Інтэрнэт Сусветная камп’ютарная сетка, здольная перадаваць даныя з аднаго пункта зямнога шара ў любы іншы.
Інфарматыка Навука, якая вывучае метады ўяўлення, захоўвання, перадачы, апрацоўкі і выкарыстання інфармацыі з ужываннем камп’ютарных тэхналогій у розных сферах чалавечай дзейнасці.
Інфармацыя Звесткі пра аб’екты і з’явы навакольнага свету.
Каманда ў алгарытме Накіраванне на выкананне канкрэтнага дзеяння.
Камп’ютар Універсальнае ўстройства для работы з данымі.
Камп’ютарная прэзентацыя Электронны дакумент, створаны на камп’ютары з дапамогай спецыяльных праграм і прызначаны для дэманстрацыі гледачам.
Камп’ютарныя тэхналогіі Тэхналогіі работы з данымі, заснаваныя на выкарыстанні камп’ютараў.
Праграмнае забеспячэнне (ПЗ) Сукупнасць усіх праграм, напісаных чалавекам для камп’ютара.
Праграмны спосаб запісу алгарытму Запіс алгарытму ў выглядзе праграмы.
Прэзентацыя Спосаб уяўлення інфармацыі з дапамогай тэхнічных сродкаў або без іх, прызначаны для дэманстрацыі чаго-небудзь новага (праекта, тавару і г. д.).
Рэдагаванне Змяненне відарыса або зместу тэксту.
Сімвал тэксту Кожная літара, лічба, знак прыпынку і г. д.
Слайд Асобная старонка прэзентацыі, прызначаная для змяшчэння матэрыялаў.
Слоўны спосаб запісу алгарытму Запіс алгарытму на натуральнай мове зносін.
Тэкставы дакумент Дакумент, інфармацыя ў якім пададзена ў выглядзе тэксту.
Тэкставы курсор Паказальнік месца ўводу сімвала.
Тэкставы рэдактар Праграма, прызначаная для работы са змесцівам электронных тэкставых дакументаў.
Фармаціраванне тэксту Змяненне формы ўяўлення тэксту.
Фрагмент відарыса Вылучаная частка відарыса.
Фрагмент тэксту Вылучаная частка камп’ютарнага тэксту.
Электронная пошта Служба сеткі Інтэрнэт, прызначаная для абмену паведамленнямі (электроннымі лістамі) з дапамогай электронных паштовых скрынь.
Электронны тэкставы дакумент Тэкставы дакумент, захаваны на дыску ў файле.
