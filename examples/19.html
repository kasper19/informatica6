Пример 19.8.

Программа построения изображения цифры 3.
  1.  С использованием команды ToPoint: 
    uses Drawman;
begin
    Field(3,4); 
    ToPoint(1,3);   PenDown; 
    ToPoint(2,3);  // 1-2
    ToPoint(1,2);  // 2-3 
    ToPoint(2,2);  // 3-4
    ToPoint(1,1);  // 4-5 
    PenUp;
end.
  2.  С использованием команды OnVector: 
    uses Drawman;
begin
    Field(3,4);
    ToPoint(1,3);   PenDown;
    OnVector(1,0);  // 1-2
    OnVector(-1,-1);// 2-3
    OnVector(1,0);  // 3-4
    OnVector(-1,-1);// 4-5
    PenUp;
end.
